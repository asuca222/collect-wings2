🔥 Firebase 設定
Realtime Database でデータを同期します。Firebase Console でプロジェクトを作成し、設定情報を入力してください。
設定手順
1. Firebase Console でプロジェクトを作成
2. Realtime Database を作成（テストモードで開始）
3. プロジェクト設定 → 「アプリを追加」→ Web アプリ
4. 表示された設定値をコピーして以下に貼り付け
⚙ Settings
旧サイト（asuca222.github.io/collect-wings-dead）のブラウザで
F12 → Console タブを開き、下のコマンドを実行してコピーしてください。
JSON.stringify({cons:JSON.parse(localStorage.getItem('cw_v3')||'[]'),lib:JSON.parse(localStorage.getItem('cw_lib_v1')||'[]'),vcap:JSON.parse(localStorage.getItem('cw_vcap_v1')||'null')})
